# 喝水提醒 macOS App

一个可爱的菜单栏喝水提醒工具：

- 每天 10:00 到 18:00 之间，每隔 1 小时弹出一次提醒。
- 水杯出现在屏幕左上角，带弹跳、气泡和柔和放大动效。
- 点击“我喝啦”后提醒消失。
- 如果不点击，每 30 秒会变大一点，直到接近铺满屏幕。
- 其他时间会自动隐藏，不再弹出。

## 构建

```bash
./scripts/build-app.sh
```

构建完成后应用会出现在：

```text
dist/WaterCupReminder.app
```

## 使用

双击 `dist/WaterCupReminder.app` 启动。菜单栏会出现一个水滴图标，可以手动触发提醒、隐藏提醒或退出应用。
